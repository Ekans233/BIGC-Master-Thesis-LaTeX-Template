# BIGC-Master-Thesis-LaTeX-Template
如题所示，这是一份由我自己和LLM总结并生成的一套完整的LaTeX硕士学位论文模板（因为学校不提供）。可以做到开包即用，格式没有任何问题，并且论无忧查重万字错误率为0。

## 背景

本人作为机械工程的研究生，写论文时经常会有图片表格以及大量公式，而这些元素word有时候不能很好的处理，于是我选择了使用LaTeX。但是我找遍内外网都没有找到可以使用的LaTeX模板，于是使用LLM读取学校的撰写规范并撰写了一版。


## 用法

把上传下来的BIGC_LatexTemplate压缩包打开，里面有个readme文档，聪明的小朋友一看就会。
