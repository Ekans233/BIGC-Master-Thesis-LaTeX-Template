## 背景

本人作为机械工程的研究生，写论文时经常会有图片表格以及大量公式，而这些元素word有时候不能很好的处理，于是我选择了使用LaTeX。但是我找遍内外网都没有找到可以使用的LaTeX模板，于是使用LLM读取学校的撰写规范并撰写了一版。
如果学校你想招安我的模板的话记得给我点个star
